シン覚え書ヘルプ
02/26/99 Rei S. <reis@hempseed.com>

このアーカイブには、シン覚え書のヘルプファイル SOboe.hlp のソース一式が含ま
れています。一通り表示確認をしたつもりですが、もしリンク不正などの問題があ
りましたら、Rei にご連絡ください。

各 RTF ファイルはかなり細かな書式設定を行っていますが、これにはコンパイル時
の不具合を回避する目的も含んでいます。
また、Word 97 で読み込んだ場合、稀にフォント情報が狂うことがあるようです。
そのまま保存してコンパイルすると、やはり予期しない結果になる場合があります。

これらの不具合は作者の環境によるものかもしれませんが、再編集/上書き後のコン
パイル結果は保証できないことをご了承ください。

ヘルプ作成環境
+ Microsoft(R) Word 97
+ Microsoft(R) Help Workshop 4.03.0002
